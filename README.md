# E-Commerce Sales Insights System

## 📊 Project Overview
This project is a SQL-based E-Commerce Analytics System designed to analyze sales performance, customer behavior, and marketing efficiency.

## 🛠️ Tech Stack
- SQL Server / PostgreSQL
- SQL (Joins, Aggregations, CTEs)
- Data Analytics Concepts

## 📌 Key Features
- Revenue Analysis
- Average Order Value (AOV)
- Customer Segmentation
- Product Performance
- Marketing ROAS Analysis

## 📈 Business Insights
- Identifies top-selling products
- Tracks customer lifetime value
- Measures marketing effectiveness
- Analyzes sales trends
